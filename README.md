# Data Structures & Algorithms in Python

A curated collection of Data Structures and Algorithms problems solved in Python, organized by interview patterns and problem-solving techniques.

The goal of this repository is not only to store solutions but also to build pattern recognition for coding interviews. Problems are grouped by the underlying algorithmic concept, making revision easier and helping identify when to apply a particular technique.

---

## Topics Covered

- Arrays & Strings
- Sliding Window
- Two Pointers
- Hashing
- Binary Search
- Stack & Queue
- Heap / Priority Queue
- Linked Lists
- Trees & Binary Search Trees
- Graphs (BFS, DFS, Topological Sort)
- Greedy Algorithms
- Dynamic Programming
- Backtracking
- Bit Manipulation
- Math & Number Theory
